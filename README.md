# SecureTransparentDataExchange Backend

Backend server-side implementation of the master's thesis project:

**Improvement of security and transparency in data exchange procedures applying blockchain and identity based encryption technologies**

## Description
This repository contains the backend implementation of the proposed secure logistics platform.

The backend provides:
- user authentication and authorization
- two-factor authentication
- shipment and parcel management
- blockchain-based immutable transaction logging
- audit logging
- AI-based anomaly detection and prediction
- payment processing integration
- notification services
- WebSocket communication
- administrative security monitoring

## Technology Stack
- ASP.NET Core (.NET 9)
- Entity Framework Core
- SQL Server
- JWT Authentication
- WebSockets
- Stripe API
- AI / ML.NET
- Blockchain logging module

## Repository Purpose
This repository is provided for academic review of the master's thesis implementation.
